# Yahoo-MCP




### Installation

Clone the repository:

```bash
git clone https://github.com/yahoo-mcp/Yahoo-MCP.git
cd Yahoo-MCP
```

### Setup

```bash
# Create a virtual environment 
uv venv 

# Activate it source .venv/bin/activate

# Install the dependencies
uv sync

# Run the agent 
uv run agent.py
```
